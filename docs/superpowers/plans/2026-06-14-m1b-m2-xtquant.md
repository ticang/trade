# M-1b/M2 xtquant 通道 + 模拟盘主线 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development. Checkbox tracking. 实现与审查分离。**xtquant Windows-only**：本机 macOS 只能写适配器代码（lazy import）+ mock 测；live xtquant 连接须 Windows + QMT 客户端登录态。凭证读 .env（gitignored）。

**Goal:** 交付 §4.6 执行层 + §4.1.1 行情网关的 QMT/模拟实现：MarketDataGateway 抽象（订阅/历史/PIT bar_at/事件总线/去重/背压/线程桥接）、QmtBroker 适配器（per-account，xtquant lazy import，on_fill 异步）、SimBroker 实盘路径集成、每日对账、DuckDB 单写进程交接。M-1b 探测项（订阅频率/回调线程/下单延迟/崩溃恢复）以代码+mock 就位，live 验证标 Windows。

**Architecture:** 接口先行——MarketDataGateway/Broker 为 Protocol。xtquant 仅在 QmtBroker/QmtGateway 内 lazy import（顶层不依赖，macOS 可测其余全部）。xtquant 回调在内部线程，固定 `loop.call_soon_threadsafe` 桥接 asyncio。on_fill 异步回调作三态统一成交入口（禁同 bar 查 status 反模式）。对账：盘后 Broker 查询 vs 本地订单簿（order_event 重放），差异落 audit。

**Tech Stack:** Python 3.11+ · asyncio · 复用 M0/M0.5/M1/M1.5。**xtquant 仅 lazy import（Windows 运行时）**

**关联：** §4.1.1（行情网关）、§4.6（执行层）、§4.6.2（订单状态机/on_fill）、§4.6.4（对账）、§4.7.5（实盘-回测一致性）、§11 M-1b/M2

**前置：** M0+M0.5+M1+M1.5+M3 完成（main）。本计划在 `feat/m1b-m2-xtquant`。

**验收标准（设计 §11 M2）：**
1. QMT 模拟连续 20 交易日（按日历）跑通信号→下单→持仓→对账闭环（mock 行情，标 Windows-only live）
2. 对账差异 < 0.1%
3. 多账户隔离正确
4. M-1b 探测项（订阅/回调/延迟/崩溃恢复）代码+mock 就位（live 标 Windows）

## File Structure

```
quant/
├── gateway/
│   ├── __init__.py
│   ├── base.py               # MarketDataGateway Protocol（subscribe/history/bar_at(PIT)/publish）
│   ├── thread_bridge.py      # xtquant 内部线程 → asyncio call_soon_threadsafe
│   ├── dedup.py              # bar 去重 (symbol,freq,trade_ts) 唯一键
│   ├── backpressure.py       # 队列深度阈值降频/合并/丢弃+告警
│   └── qmt.py                # QmtGateway（xtquant lazy import，macOS 不可 live，mock 测）
├── execution/
│   ├── __init__.py
│   ├── broker.py             # Broker Protocol + OrderStatus 状态机 + on_fill 模板
│   ├── sim_broker_live.py    # SimBroker 实盘路径包装（复用 M1 SimBroker，is_synchronous=True）
│   ├── qmt_broker.py         # QmtBroker（xtquant lazy import，per-account，on_fill 异步）
│   ├── order_book.py         # 本地订单簿（order_event 重放恢复）
│   └── reconcile.py          # 每日对账（Broker vs 本地，差异落 audit）
tests/quant/test_gateway_*.py / test_execution_*.py（扁平）
```
